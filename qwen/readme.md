# 基于qwen模型的多模态SAE训练和特征反演

## SAE训练
- 因为后面要做特征反演，因此训练时对于特征不能取mean压缩，需要训练时把图片的每个patch作为一个训练数据来训练，因此提取出来的内容就是每个训练数据的图片-文本特征向量对
- 但训练之前要先进行图片-文本特征向量对齐，VL-SAE原文因为进行了压缩，图片-文本对都被压缩到相同的隐藏维度，但是当前方法没做压缩，图片文本的维度是不同的，无法直接计算InfoNCE损失来训练编码器，因此我们采取FILIP集合对齐法，每个图片patch都去寻找最相似的文本patch，文本也寻找最相似的图片patch，记录每个的最大相似度，然后再把这些最大相似度加起来作为得分来训练用来做图像文本对齐的辅助encoder
- 最后训练SAE的时候是把得到的图片文本对全部打散喂给基于欧氏距离的encoder，进行topk训练
